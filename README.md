# Lab 2 - UI Layout Practice

## Purpose + Objective: 
To demonstrate proficiency in creating a .NET MAUI application interface using layout controls, view controls, resources, event handlers, and page controls. 

Consider selecting a topic or theme when designing your interface. You can build your interface to reflect information regarding your favorite hobby or restaurant, or you can center your interface around a historical event or person. 

Make sure your application is visually appealing and user-friendly.

This lab contributes to fulfilling the weekly objectives listed below.

- Construct a layout using permissible layout panels offered in .NET MAUI

- Identify the usefulness of implementing resources at the application level and using files App.xaml and App.xaml.cs


## Instructions:
- Create a new .NET MAUI application in Visual Studio code.
- Use three or more page controls such as ContentPage, NavigationPage, and TabbedPage to create different pages for your application.
  - Demonstrate page navigation by implementing functionality to navigate to different pages within your application
  - Consider using either a FlyoutPage or TabbedPage to navigate between different pages in your application.
- Use a layout control to organize and arrange the view controls of your application.
- Include six or more view controls such as labels, buttons, text boxes, and images in your application.
- Define resources at the application level, such as colors, fonts, and styles.
- Make use of two or more event handler methods to add basic functionality to your application.

- Implement both Images and icons into your application to add visual appeal.
  -  Sources for Images and Icons are listed below
     - Images:
       - [Pexels.com](https://pexels.com)
       - [Unsplash.com](https://unsplash.com)
       - [Burst.shopify.com](https://burst.shopify.com)
     - Icons:
       - [feathericons.com](https://feathericons.com/)
       - [fontawesome.com/icons](https://fontawesome.com/icons)
       - [Orioniconlibrary.com](https://www.orioniconlibrary.com/all-icons)


## Grading Criteria:
* Correct use of layout control to organize and arrange view controls (2  points)
* Inclusion of six or more view controls (2 points)
* Correct definition of resources at the application level (1 point)
* Use of two or more event handler methods (2 points)
* Inclusion of three or more page controls (3 points)
